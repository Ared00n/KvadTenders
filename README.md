# Квадрат - Сайт компании

Веб-сайт компании "Квадрат" для поставок товаров через электронные торги.

## Запуск локально

```bash
go run main.go
```

## Развертывание на Render

Проект настроен для автоматического развертывания на Render.com

### Команды сборки:
- Build Command: `go build -o main main.go`
- Start Command: `./main`

### Переменные окружения:
- PORT - автоматически устанавливается Render
